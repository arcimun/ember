Hi Groq team,

I built Ember — a free, open-source macOS voice-to-text app powered by Groq's Whisper API + Llama 3.3.

The end-to-end latency is ~1.7 seconds (0.7s Whisper + 1s LLM correction), which is the fastest voice-to-text experience I've seen on macOS. It's a great showcase for Groq's inference speed.

- GitHub: https://github.com/arcimun/ember
- Install: `brew install --cask arcimun/tap/ember`
- MIT licensed, ~1,500 LOC Swift

Would love to:
1. Be featured in Groq's community showcase / blog
2. Add "Powered by Groq" badge (already planned)
3. Discuss any developer program / extended free tier for Ember users

The app has a WebGL2 plasma overlay that makes dictation feel magical — happy to provide demo videos/GIFs for any content you'd want to create.
